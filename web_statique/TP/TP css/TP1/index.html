MANIPULATION DES BALISES HTML ET DES SÉLECTEURS CSS

Voici venu le moment d'expérimenter les usages des différente sbalises HTML et de composer une page de présentation simple, claire, normalisée et évolutive.

Cette page vise nottament à monter la distinction entre

  • les éléments de bloc (div, h1, ul ...)
  • les éléments inline (img, a, span ...)

Vérifiez que vous avez bien indiqué l'encodage U T F-8 grâce à la balise <meta>. (Attention, pour que la phrase précédente s'affiche cerrectement, il est nécessaire d'utiliser de entités HTML :cherchez sur le web de quoi il s'agit!)
